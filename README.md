# Projeto-AtomicAdventures
Trata-se de um jogo inspirado no detetive, focado em consolidar conhecimentos de matemática básica para o estudo da química. Com desafios envolventes e uma narrativa investigativa, os jogadores exploram conceitos científicos enquanto aprimoram concentração e habilidades analíticas. Ideal para tornar o aprendizado divertido e eficaz!
